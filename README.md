# Club Alianza Catamarca

Landing page creada como parte del TP1 del Modulo de Frontend con React del Nodo Tecnológico de Catamarca

## Pasos para instalar el proyecto

1. Clona el repositorio
```bash
git clone git@github.com:AxelClosas/sprint-1-tp1-react.git
```

2. Instala las dependecias con pnpm
```bash
pnpm install
```

3. Inicia el servidor de desarrollo
```bash
pnpm run dev
```
